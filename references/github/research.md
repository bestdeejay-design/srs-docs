## 3. Ключевой артефакт: DATA.js

**Файл:** `test-task-1/DATA.js`
**Источник:** https://github.com/сервис-аналог/test-task-1/blob/master/DATA.js

Содержит полную структуру каталога товаров Агрегатор.

### Структура данных

```javascript
// Категория 1-го уровня
{
  id: 1,
  name: 'Овощи, фрукты, грибы, ягоды',
  child_categories: [
    {
      id: 18,
      name: 'Овощи',
      icon_path: '/uploads/category/image/18/Rectangle.png',
      price: { from: 5.2, to: 654.0 },
      filters: [
        { id: 9, name: 'Вид овоща', values: [{ id, name }, ...] },
        { id: 0, name: 'Бренд', values: [{ id, name }, ...] }
      ]
    },
    // ...ещё 6 подкатегорий
  ]
}
```

### Категории верхнего уровня

| ID | Название | Подкатегорий |
|---|---|---|
| 1 | Овощи, фрукты, грибы, ягоды | 7 |
| 2 | Молочные продукты, сыр и яйца | 7+ |
| 3 | (продолжение в файле) | ... |

### Фильтры по категориям

Пример фильтров для категории **Молоко и сливки**:
- **Тип:** сливки, молоко, концентрированное, безлактозное, козье
- **Тепловая обработка:** стерилизованное, ультрапастеризованное, топлёное, пастеризованное
- **Жирность:** 0–40%, разбивка по диапазонам
- **Фермерский продукт:** булевый
- **Бренд:** ~50 брендов (Valio, Простоквашино, Домик в деревне и др.)

## 5. Что из этого полезно для проекта

| Артефакт | Польза |
