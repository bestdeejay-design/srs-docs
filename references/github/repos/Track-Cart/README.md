🛒 Track-Cart

Track & Cart is a full-stack grocery delivery platform built with Next.js 15, MongoDB, NextAuth, and Socket.io.
It provides real-time order tracking, live chat with delivery partners, AI-powered assistance, and a modern animated UI.

The platform focuses on fast performance, real-time communication, and a smooth user experience.

🚀 Features
📦 Real-Time Order Tracking

Live map tracking of delivery orders

Real-time location updates

Track delivery partner movement instantly

💬 Real-Time Chat

User ↔ Delivery partner chat

Built using Socket.io

Instant messaging without page refresh

🔐 Authentication

Secure login using NextAuth

Session-based authentication

Protected routes for users

🛍 Grocery Ordering System

Browse grocery products

Add products to cart

Place and manage orders

🤖 AI-Powered Assistance

Smart assistance to help users

Improve shopping and ordering experience
