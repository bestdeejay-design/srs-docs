# Документация продукта

Репозиторий для хранения документации проекта.

## Структура

```
docs/
├── README.md                          # Этот файл
├── ПЛАН_ОПТИМИЗАЦИИ_АЙГУДС.md         # План миграции на микросервисы
├── ШАБЛОН_СПЕЦИФИКАЦИИ_ПРОДУКТА.md     # Шаблон для описания бизнес-процессов
```

| Файл | Описание |
